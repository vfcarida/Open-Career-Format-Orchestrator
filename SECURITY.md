# Security Policy

## Supported Versions

Only the latest release of the Open Career Format Orchestrator (OCF) is actively supported for security updates.

| Version | Supported |
| ------- | --------- |
| >= 0.1.0| Yes       |

## Reporting a Vulnerability

We take the security of user career data and automated credentials seriously. If you discover a vulnerability, please report it to us following these guidelines:

1.  **Do Not File Public Issues**: Avoid filing public issues for potential security vulnerabilities.
2.  **Submit a Security Report**: Send a detailed description of the vulnerability, including steps to reproduce, to the project maintainers (security@example.com).
3.  **Coordinated Disclosure**: We aim to resolve vulnerabilities within 30 days of receipt before releasing details publicly.

## Safe Harbor

We support security research conducted in good faith. If you discover a security issue while running OCF locally on your sandbox machines or static fixtures, we will coordinate resolution without legal action, provided you adhere to this policy.
