# Architecture Decision Record: Use OKF

## Context

Agents need context to understand the project architecture. Passing random READMEs causes context flooding and hallucination.

## Decision

We will use the Open Knowledge Format (OKF) with Frontmatter to structure all architectural decisions, allowing AKCP to selectively inject these into the agent's context window.

## Consequences

- Requires strict schema enforcement for markdown files.
- Reduces token costs by 40% during agent codebase reasoning.
