# Suggested GitHub Labels

The following labels are recommended for organizing AKCP issues and PRs. Maintainers should create these in the GitHub repository settings.

## Workstream Labels

| Label | Color | Description |
|-------|-------|-------------|
| `workstream: compiler` | `#0075ca` | Changes to the OKF compilation pipeline |
| `workstream: control-plane` | `#7057ff` | Changes to the MCP Gateway, policies, HITL |
| `workstream: connectors` | `#008672` | New or updated source connector plugins |
| `workstream: policies` | `#e4e669` | Policy pack authoring and governance |
| `workstream: evals` | `#d876e3` | Evaluation scenarios and benchmarks |
| `workstream: docs` | `#e1d0fb` | Documentation, spec, and tutorial improvements |
| `workstream: examples` | `#fef2c0` | Domain example bundles |

## Priority Labels

| Label | Color | Description |
|-------|-------|-------------|
| `priority: critical` | `#b60205` | Blocking bug or security issue |
| `priority: high` | `#e11d48` | Important feature or non-blocking bug |
| `priority: medium` | `#f97316` | Normal priority work |
| `priority: low` | `#94a3b8` | Nice-to-have or future work |

## Status Labels

| Label | Color | Description |
|-------|-------|-------------|
| `good first issue` | `#7fc97f` | Suitable for new contributors |
| `help wanted` | `#159818` | Maintainers are actively seeking contributors |
| `needs: RFC` | `#c2e0c6` | This change requires a formal RFC |
| `needs: spec-update` | `#bfd4f2` | Implementation requires a spec change |
| `status: blocked` | `#e11d48` | Blocked by an external dependency |
| `status: in-progress` | `#0052cc` | Actively being worked on |

## Type Labels

| Label | Color | Description |
|-------|-------|-------------|
| `type: bug` | `#d73a4a` | Something is not working as specified |
| `type: feature` | `#a2eeef` | New functionality |
| `type: security` | `#b60205` | Security vulnerability or concern |
| `type: performance` | `#fbca04` | Performance regression or improvement |
