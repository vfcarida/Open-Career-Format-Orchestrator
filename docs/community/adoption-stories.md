# AKCP Adoption Stories

This document captures real-world adoption patterns and success stories. It serves both as inspiration and as a template for new adopters to share their experience.

> **Want to share your story?** Open a PR adding a new entry following the template below.

---

## Template

```markdown
## [Company/Team Name] — [Domain]

**Date:** YYYY-MM
**Team Size:** (e.g., 2 engineers, 1 SME)
**Domain:** (e.g., Software Engineering, IT Operations, Customer Support)

### Problem
What challenge were you solving? What were agents doing wrong before AKCP?

### Solution
How did you use AKCP? What domain did you compile? Which targets did you use?

### Results
- Metric 1: (e.g., MTTR reduced from 45min → 12min)
- Metric 2: (e.g., Context token usage reduced by 62%)
- Qualitative: (e.g., "Agents stopped suggesting rollback steps that no longer apply.")

### Key Learnings
What surprised you? What would you do differently?
```
