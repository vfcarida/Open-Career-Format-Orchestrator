# @ocf/dashboard

A browser-native Single Page Application (SPA) designed to visualize and inspect your local Open Career Format (OCF) directory bundle.

## Features

- **Directory Picker Integration**: Connect directly to your local `.okf/` folder.
- **Kanban Board Pipeline**: Track active and historical applications.
- **Force connection Graph**: Interactive visualization of candidate competencies mapped against past companies using D3.js.
- **In-Memory Validation**: Highlights YAML frontmatter conformance issues without sending any data to external servers.

## Development

```bash
# Run the vite development server
pnpm dev
```
