# Help Article: How to process refunds

**Policy:**
Refunds can only be issued within 30 days of the original purchase date.
Agents must never autonomously issue a refund without prior Human-In-The-Loop approval.

**Procedure:**
1. Verify the purchase date is within 30 days.
2. Confirm the reason for the refund.
3. Submit a refund request using the `issue_refund` capability.
4. Wait for a Tier 2 manager to approve the request via the Control Plane.
5. Notify the customer once approved.
